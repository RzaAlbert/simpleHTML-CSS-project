Podstawy HTML

Podstawowe znaczniki HTML to:

  • <html> - korzen dokumentu
  • <head> - nagłowek strony z informajami dla przegladarek
  • <body> - tresc dokumentu
  • <div> - element blokowy
  • <p> - akapit
  • <ol> <ul> i <li>- elementy listy wypunktowanej lub numerowanej
  • <img> - obrazek
  • <a> - hiperłącze
